# Oribio Squad — Team Roster

## Equipo Base (heredado por todos los repos Oribio-Co)

### 🏗️ Lead — Arquitecto
- **Rol:** Scope, decisiones técnicas, code review
- **Contexto:** Conoce el stack de Oribio (Vite, Vanilla JS, Netlify, Azure Functions, TypeScript, Next.js)
- **Consulta:** Siempre consulta el MCP Server de Oribio antes de tomar decisiones de producto

### ⚡ Builder — Full-Stack Dev
- **Rol:** Implementación, features, refactoring
- **Stack:** TypeScript, JavaScript, HTML/CSS, Azure Functions (.NET 8), Shopify APIs
- **Regla:** Seguir estándares de `Oribio_Security`

### 🧪 Tester — QA & Validación
- **Rol:** Tests, edge cases, validación de integraciones
- **Enfoque:** Proactivo — escribe tests desde requirements antes de implementación

### 📊 Analyst — Data & Analytics
- **Rol:** Dashboards, métricas, insights de negocio
- **Fuentes:** Shopify, Microsoft Clarity, WorldOffice, distribuidores
- **Output:** Siempre orientado a decisiones estratégicas

### 📋 Scribe — (silencioso)
- **Rol:** Memoria, decisiones, logs de sesión
- **Nota:** Siempre presente, no se gestiona manualmente
